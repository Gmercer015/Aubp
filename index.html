Welcome to the Port Of Subs Automated Bread Pull

Make the nightly bread pull even easier with this simple to use software. Customize your program to fit your stores exact needs and start speeding up your close tonight.

Downloading Automated Bread Pull is simple and easy to install. Simply download the archive your operating system supports and extract the contents into the directory of your choice. Head inside the BIN directory and create a shortcut to the application inside, drag the shortcut to the desktop and you're ready to start the bread pull!

Want to Help Develop?

Aubp is open source software, meaning the code is open to the public and completely free to use. If you would like to help maintain and add features to Aubp then GitHub is where it's at. Head on over to my source code and Fork my repository to start your own work, then type in:

$ git clone https://github.com/username/breadPull.git
$ git remote add upstream https://github.com/Gmercer015/breadPull.git
$ git fetch upstream

You can find more information about working with GitHub at help.github.com

Enjoy a Highly Customizable Enviroment to Fit Your Every Need

Just about every aspect of Bread Pull can be modified to fit your stores specific needs. Constants that are able to be edited include(but are not limited to):

  • sticks per box
  • round sticks to nearest N
  • force round up
  • daily sailes
  • bread cost

Built by Employees, for Employees

After switching shifts to a weekly closing schedule Grant saw the opportunity to help out employees everywhere by implimenting an application to automate the stores nightly bread pull.
